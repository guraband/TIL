# TIL (Today I Learned)

프로그래밍 관련 오늘 학습한 것을 기록하는 저장소입니다.

## 📁 구조

| 경로 | 설명 |
|------|------|
| `TIL.md` | 현재 작성 중인 TIL (최신 학습 기록) |
| `til_old/` | 월별로 아카이브된 과거 TIL |
| `images/` | TIL에 첨부하는 이미지 |
| `pdf/` | 참고 자료 PDF |

### 아카이브 구조

- `til_old/2023/` ~ `til_old/2025/` : 연도별 폴더
- `TIL_YYYYMM.md` : 해당 연·월의 TIL (예: `TIL_202512.md`)

## ✏️ 작성 방식

- 날짜별로 구분하여 오늘 배운 내용, 참고한 자료, 링크 등을 간단히 기록합니다.
- 개발, 유튜브, 도서, 토이 프로젝트 등 다양한 주제를 포함합니다.

## 📌 참고

- 현재 TIL은 `TIL.md`에 이어서 작성하고, 필요 시 `til_old/`로 월별 정리합니다.
